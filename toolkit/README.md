# Toolkit Content

This folder contains the practical working materials for the EU AI Act Toolkit.

| Folder | Purpose |
|---|---|
| checklists/ | Readiness, procurement, and internal-use checklists |
| templates/ | Inventory, screening, vendor, policy, literacy, incident, and evidence templates |
| starter-pack/ | Fast-start package for SMEs beginning readiness work |
| vendor-pack/ | Vendor assessment materials for AI-enabled SaaS procurement |
| sector-packs/ | Sector and use-case specific guidance |
| examples/ | Fictional examples and sample evidence packs |

These materials are grouped under `toolkit/` to keep the repository root compact and easier to scan.
