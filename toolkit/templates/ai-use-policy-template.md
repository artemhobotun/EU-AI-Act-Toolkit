## 4. Prohibited Uses (Examples)

- generating or sending customer communications without human review (unless explicitly approved)
- using AI to make final decisions in high-impact contexts without defined oversight
- bypassing security controls or exporting data to unapproved tools

## 5. Data Rules (Critical)

Do not input the following into third-party AI tools unless explicitly approved:

- passwords, API keys, secrets
- confidential business information (contracts, pricing, strategy)
- sensitive personal data or special category personal data
- children's personal data

If you allow uploads to certain tools, list the conditions here (approved tools, encryption, retention, DPA, etc.).
