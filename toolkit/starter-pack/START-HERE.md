# Start Here

If you only have 30 minutes:

1. Open the [30-minute readiness self-assessment](printable/docs/guide/30-minute-readiness-self-assessment.md).
2. List your AI tools.
3. Mark whether personal data is involved.
4. Identify tools affecting people.
5. Escalate sensitive or high-impact use cases.
6. Assign an owner.
7. Store evidence internally.
