# One-Page Executive Checklist

For founders, management, and SME operators.

- [ ] Inventory the AI tools and features in use.
- [ ] Assign an owner for each important system.
- [ ] Confirm what data is involved.
- [ ] Review the vendor before purchase or rollout.
- [ ] Require human review for important outputs.
- [ ] Publish internal AI use rules.
- [ ] Record AI literacy or onboarding.
- [ ] Keep an incident log.
- [ ] Set a maintenance review cadence.
- [ ] Escalate sensitive or high-impact use cases.
