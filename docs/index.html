How to Use This Toolkit

  1. Define your AI inventory: List the AI tools you use and who manages each one.
  2. Run a readiness check: Screen each tool for documentation gaps, data processing concerns, and vendor clarity.
  3. Document internally: Create simple governance records (policies, owner checklists, incident processes).
  4. Escalate where needed: For high-risk cases or unclear vendor practices, escalate to qualified legal, privacy, or security review.
