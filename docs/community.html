Community & Contribution

The EU AI Act Toolkit is open source and improves through community feedback. Every contribution — from a typo fix to a new template — makes the resource more useful for everyone.

Ways to Get Involved

How to Contribute

Different contribution paths depending on your time, interest, and expertise.

📝

Improve Wording

Found a typo, an unclear phrase, or a better way to explain something? Small improvements in clarity have a big impact on usability across the whole toolkit.

Contributing Guide
💡

Suggest a Template

Think a new template, checklist, or reference document would be useful? Open a template request issue and describe the use case it would serve.

Create Issue
🎯

Add a Use Case Example

Have a realistic example of how a governance or procurement scenario plays out? Use case examples help other organisations understand what to expect.

Suggest Example
🔗

Update Source References

Spotted an outdated reference, a broken link, or a new authoritative source that should be included? Let us know.

Report Update
🐛

Report a Bug or Error

Found a factual error, broken link, or formatting issue? Open a GitHub issue with as much context as possible.

GitHub Issues
💬

Share General Feedback

General thoughts on the toolkit's usefulness, organisation, or gaps? Feedback on what is actually helping people is always welcome.
