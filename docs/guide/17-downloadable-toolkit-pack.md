# Downloadable Toolkit Pack

## Purpose

SMEs can download or copy templates and maintain completed documents internally.
For vendor review work, see the optional [Vendor Assessment Pack](../vendor-pack/README.md).

## How to download

- GitHub "Code" -> "Download ZIP"
- clone repo
- copy individual templates

## Recommended internal folder structure

```text
AI-governance/
inventory/
screening/
vendors/
policy/
incidents/
literacy/
reviews/
```

## How to use the templates

1. Make a copy.
2. Add date and version.
3. Assign an owner.
4. Complete internally.
5. Store securely.
6. Review periodically.

## What not to put in GitHub issues

- personal data
- confidential business data
- customer data
- contracts
- security details
- trade secrets

## Version tracking

Record which toolkit release or commit was used when completing internal documents.
