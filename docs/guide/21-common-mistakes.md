# Common Mistakes

- treating all AI tools as low-risk
- not keeping an inventory
- no owner assigned
- using sensitive data in public AI tools
- no vendor documentation
- no human review
- no AI literacy records
- confusing marketing claims with compliance evidence
- assuming a template guarantees compliance
- forgetting to update documents

These mistakes are common in small teams because AI adoption often starts informally. The toolkit is meant to reduce that drift.
