# Maintainer Content Style Guide

Use this guide to keep future contributions consistent.

- Use practical English
- Avoid legal overclaiming
- Use "may require review"
- Link to source notes
- Keep the SME user in mind
- Do not include confidential data
- Use templates and checklists
- Prefer short sections and tables
- Add a disclaimer where needed

The goal is to keep the toolkit useful as a starting point for readiness work and documentation support.
