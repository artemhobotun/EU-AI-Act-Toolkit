Learning Paths

If you're new to AI governance...

  1. Read the home page intro (5 min)
  2. Run the 30-minute readiness self-assessment (30 min)
  3. Check the FAQ (10 min)
  4. Review the glossary for unfamiliar terms (20 min)
  5. Use the decision tree for your first tool review (15 min)

If you need to buy or renew an AI SaaS tool...

  1. Review the Vendor Assessment Pack overview (10 min)
  2. Check vendor examples to see how others approached it (15 min)
  3. Fill out the vendor questionnaire (varies)
  4. Use the comparison matrix if evaluating multiple vendors (varies)
  5. Document your decision with a decision record (10 min)
